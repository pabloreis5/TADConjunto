# TADConjunto
AEDS1 - Algoritmo implementado utilizando Lista, afim de realizar operações com conjuntos.

## Funcionalidades
* void FCVazio - Inicia determinado conjunto com 0 elementos, ou seja, vazio.
* int CVazio - Verifica se determinado conjunto está vazio, retornando 1 para true, 0 para false.
* void CInsere - Insere elemento informado pelo usuário em determinado conjunto.
* void CLeitura - Realiza a leitura e insere. Implementada para facilitar a inserção de variados elementos, utiliza-se da função CInsere.
* void CImprime - Imprime todos os elementos de um determinado conjunto.
* void UniaoConjuntos - Realiza a união de dois conjuntos e os insere em um terceiro conjunto resultante.
* void IntersecaoConj - Realiza a interseção de dois conjuntos e os insere em um terceiro conjunto resultante.
